# Project guidance for Claude Code

## Git commits

- Do **not** add `Co-Authored-By: Claude ...` trailers to commit messages.
- Do **not** add `Claude-Session:` trailers to commit messages.
- Do **not** add "🤖 Generated with Claude Code" or similar attribution to commit messages or PR bodies.

Keep commit messages clean: a concise subject line and body describing the change only.
