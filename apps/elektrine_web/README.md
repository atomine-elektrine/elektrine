# Elektrine Web

Phoenix web interface app for the Elektrine umbrella.

## What this app owns

- Endpoint, router, plugs, channels, and API controllers
- LiveView UI for timeline, chat, email, settings, admin, and profile flows
- Web adapters for ActivityPub, messaging federation, DAV, and JMAP endpoints

## Notes

- Depends on `elektrine` and feature apps in the umbrella.
- Templates, components, and LiveViews are in `apps/elektrine_web/lib/elektrine_web`.
