# Password Manager Extension

The browser extension is a client artifact, not part of the server deploy.

Canonical path:

- `clients/password-manager-extension/`

For local development, load the unpacked extension from `clients/password-manager-extension/`.

The extension accepts production servers over `https://` and local loopback servers over
`http://localhost`, `http://127.0.0.1`, or `http://[::1]`.
